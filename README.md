# Rate it!
###### Виджет для оценки задач команды тестирования
## Установка:
1. Скачать репозиторий
2. Открыть [chrome://extensions/]()
3. Включить режим разработчика
4. Нажать "Установить распакованное расширение"
5. Выбрать папку с приложением
